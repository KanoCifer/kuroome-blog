📚 HTML5 备忘单

开发者必备的 HTML5 核心特性、语义化标签、表单增强、媒体标签及常用 API 快速查询指南

1. 文档基础结构与 Meta 标签

标准文档模板

<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>页面标题</title>
</head>
<body>
    <!-- 内容 -->
</body>
</html>

常用 Meta 标签

标签/属性 说明
<meta charset="UTF-8"> 定义字符编码
<meta name="viewport"> 移动端视口配置（响应式设计必备）
<meta name="description"> 页面描述（SEO 优化）
<meta name="keywords"> 页面关键词
<meta http-equiv="X-UA-Compatible" content="IE=edge"> 强制 IE 使用最新渲染引擎

2. HTML5 常用标签速查

2.1 文档结构与元信息

标签 说明
<html> 文档根元素（常用 lang）
<head> 元数据容器
<title> 页面标题
<meta> 元信息（编码、视口、描述等）
<link> 关联外部资源（CSS、图标等）

<style> 内联样式

2.2 语义化结构

标签 说明

<footer> 页脚或区块尾部
<address> 联系信息
<figure> / <figcaption> 图表/媒体与标题
<time> 日期/时间（可配合 datetime）

<sub> / <sup> 下标/上标

<cite> 引用来源
<ruby> / <rt> / <rp> 注音标注
<bdi> / <bdo> / <wbr> 文字方向隔离/覆盖/建议换行点

2.4 列表

3. 表单增强 (Forms)

新增 Input 类型

  • email: 自动验证邮箱格式
  • url: 自动验证 URL 格式
  • tel: 电话号码输入（移动端弹出数字键盘）
  • number: 数字输入（支持 min, max, step）
  • range: 滑块控件
  • date / month / week / time: 日期时间选择器
  • color: 颜色选择器
  • search: 搜索框

常用表单属性

属性 说明
placeholder 输入框提示文字
required 必填字段
autofocus 页面加载时自动获取焦点
autocomplete 开启/关闭自动填充 (on/off)
multiple 允许上传多个文件或输入多个邮箱
pattern 使用正则表达式验证输入
list 引用 <datalist> 定义的预设选项

4. 多媒体标签

视频与音频

<!-- 视频播放 -->
<video width="640" height="360" controls poster="thumbnail.jpg">
    <source src="movie.mp4" type="video/mp4">
    您的浏览器不支持 video 标签。
</video>

<!-- 音频播放 -->
<audio controls>
    <source src="music.mp3" type="audio/mpeg">
</audio>

常用属性: controls (显示控件), autoplay (自动播放), loop (循环), muted (静音)。

5. 全局属性与无障碍 (ARIA)

常用全局属性

  • class / id: 样式与脚本标识
  • data-*: 自定义数据属性（用于存储私有数据）
  • contenteditable: 使元素内容可编辑
  • draggable: 设置元素是否可拖拽
  • hidden: 隐藏元素
  • title: 鼠标悬停时的提示文本

ARIA 基础 (无障碍)

  • role="...": 定义元素的角色（如 role="button", role="navigation"）
  • aria-label: 为没有文本内容的元素提供标签
  • aria-hidden="true": 对辅助技术（如屏幕阅读器）隐藏元素

6. 核心 API 概览

Canvas (画布)

用于通过 JavaScript 绘制 2D 图形。

const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');
ctx.fillStyle = '#FF0000';
ctx.fillRect(0, 0, 150, 75); // 绘制红色矩形

Web Storage (本地存储)

  • localStorage: 持久化存储，除非手动删除，否则永不过期。
  • sessionStorage: 会话级存储，关闭浏览器标签页后清除。 
// 存储数据
localStorage.setItem('username', 'Sisyphus');
// 读取数据
const user = localStorage.getItem('username');

Geolocation (地理定位)

获取用户地理位置。

navigator.geolocation.getCurrentPosition((position) => {
    console.log(position.coords.latitude, position.coords.longitude);
});
