# ADR-0001: Record Architecture Decisions

## Status

Accepted

## Context

We need a way to record architectural decisions made in this project.

## Decision

Use Architecture Decision Records (ADRs) stored in `docs/adr/`.

## Consequences

- Decisions are documented and discoverable
- New ADRs are numbered sequentially: `NNNN-slug.md`
- Each ADR follows: Status, Context, Decision, Consequences
